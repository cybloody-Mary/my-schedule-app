# my-schedule-app
个人日程管理工具-Personal Schedule Management Tool
📅 多视图日历
日 / 周 / 月视图自由切换，月视图高亮展示重点项目名称，点击日期可直接跳转当日日视图
⏱️ 横向可缩放时间轴
横向展示全天日程，拖动滑块实时显示时间变化，支持缩放调整精度
✅ 任务与日程管理
支持项目名称、简介、时间段、分类标签，可设 9 种自定义颜色
支持标记重点、完成状态，支持批量一键添加重复日程
🌳 日期绑定流程图
树状分支结构，可给指定日期创建流程图
日 / 周视图均提供快捷创建按钮
🚀 专注页面（番茄钟 + 闹钟）
独立纯净页面，含普通闹钟与自定义时长番茄钟，自动记忆最近 8 个常用时长
支持开始 / 暂停 / 继续
🔔 提醒与铃声
闹钟、番茄钟均支持铃声开关，内置多首提示音，支持自定义音频导入
可选择仅弹窗提醒，不播放铃声
🎵 全局背景音乐
支持导入音频、创建歌单、删除歌单（二次确认）
带可拖动进度条，支持顺序 / 随机 / 单曲循环，切歌逻辑与播放模式联动
📊 数据统计
统计每日完成数、累计完成数、分类完成量，以环形图展示
🖨️ 日程导出
支持日 / 周 / 月视图多选，导出为高清 PDF 或 PNG 图片
💾 本地存储
所有数据永久本地保存，关闭网页后重新打开可完整恢复


# 📅 Multi-View Calendar
- Freely switch between daily / weekly / monthly views.
- Highlight key project names in monthly view.
- Click a date to directly jump to its daily view.

# ⏱️ Horizontally Resizable Timeline
- Displays all-day schedules horizontally.
- Drag the slider to view real-time time changes.
- Supports zooming to adjust time precision.

# ✅ Task & Schedule Management
- Supports project name, description, time period, and category tags.
- Customizable with 9 distinct colors.
- Allows marking priorities and completion status.
- Supports one-click batch addition of recurring schedules.

# 🌳 Date-Bound Flowchart Notes
- Tree-branch structure for creating flowcharts linked to specific dates.
- Quick-create buttons available in both daily and weekly views.

# 🚀 Focus Mode (Pomodoro + Alarm Clock)
- Clean, independent interface with regular alarms and custom-duration Pomodoro timer.
- Automatically saves the last 8 frequently used durations.
- Supports start / pause / resume operations.

# 🔔 Reminders & Ringtones
- Toggle sound for alarms and Pomodoro notifications.
- Built-in multiple alert tones and support for custom audio import.
- Option for popup-only reminders without sound.

# 🎵 Global Background Music
- Supports audio import, playlist creation, and playlist deletion (with double confirmation).
- Draggable progress bar with play modes: sequential / shuffle / single repeat.
- Track switching logic synced with selected play mode.

# 📊 Data Statistics
- Tracks daily completed tasks, total completions, and category-wise progress.
- Visualized with a donut chart.

# 🖨️ Schedule Export
- Supports multi-selection in daily / weekly / monthly views.
- Exports to high-definition PDF or PNG images.

# 💾 Local Storage
- All data saved permanently in local storage.
- Full data restoration when reopening the webpage.
